Object detection technique helps in the recognition, detection, and localization of multiple visual instances of objects in an image or a video. It provides a much better understanding of the object as a whole, rather than just basic object classification. This method can be used to count the number of instances of unique objects and mark their precise locations, along with labeling. You Only Look Once, or YOLO, is a techniques for object recognition designed for speed and real-time use. We can think of an object detector as a combination of a object locator and an object recognizer. We have used YOLO for this project.
